## step1 准备工作
+  以我的手机（华为mate20）作为演示
+ 下载Auto.js安装包并安装
+ 下载脚本xduyqt.js
+ 打开手机的无障碍模式，将无障碍快捷方式设置成Auto.js，同时按下上下键3s以上打开，成功应该会出现以下提示信息
+ 将微信中的西电企业号会话置顶
****
## step2
+ 新建文件，将脚本内容复制
+ 点击运行
****
## step3
+ 大功告成，以后每次填疫情通时运行脚本就欧克了
****
**如果出现问题，重启手机后同时按上下音量键3s以上重新开启无障碍模式，关机重启后同理**
